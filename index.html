¿QUE ES ASSETS MAGER?

¿Usted se encuentra involucrado en el negocio de los bienes raíces? ¿Tiene problemas para administrar sus propiedades? ¡No busque más que aquí encontrará la solución!
Assets Mager es una aplicación web diseñada para ayudarle en esta tarea, permitiendole a usted como usuario, almacenar y administrar la información de sus propiedades, ¿Lo mejor de todo? ¡Totalmente gratis!

¿COMO FUNCIONA?

Assets Mager, como se comentó anteriormente, es un aplicativo web destinado a la administración de bienes raices, funcionando por medio de una base de datos, el usuario podrá subir la correspondiente información para poder administrarla.
Tras registrarse podrá acceder a la aplicación, por medio de esta será capaz de registrar información de sus propiedades para poder subirlas a una base de datos y con ello tener completo acceso a toda esta data.

¿COMO EMPEZAR?

Para empezar a usar la aplicación deberá dirigirse a la pagina de Assets Mager (ubicada en https://assets-mager.herokuapp.com) y dirigirse al apartado de "Registrarme" con ello podrá crear una cuenta, y una vez validada su información, será capaz de acceder a la aplicación para hacer uso de esta
Una vez dentro de la aplicación podrá comenzar a subir información a esta, unicamente requiere de dirigirse al apartado de "Agregar" y con ello puede subir información a la nube, despues podrá verla y filtrarla en base a sus criterios.
